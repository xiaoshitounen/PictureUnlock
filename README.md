# PictureUnlock
#### Android: 图案解锁

**原理：将衣服撕掉的图片和没有撕掉的图片叠放在一起，填充满屏幕。通过操作表面的图片---触摸点四周的图片像素设置为透明，达到撕掉衣服的效果。**

**学习内容：`RelativeLayout`   `代码布局`   `View的setOnTouchListener`**

**难度：新手。**
